TUGAS PRAKTIKUM SO SESI 1

Tugas dikerjakan secara berkelompok 2 anggota, boleh 3 anggota (maks) kalau satu orang tidak ada timnya.  
Anggota 1 (Edo):  
&nbsp;&nbsp;&nbsp;&nbsp;1. Buat repo baru, nama kan sesuai nama kelompok  
&nbsp;&nbsp;&nbsp;&nbsp;2. Tambahkan kolaborator repo untuk teman tim  
&nbsp;&nbsp;&nbsp;&nbsp;3. Buat branch dengan format penamaan (nama/nim)  
&nbsp;&nbsp;&nbsp;&nbsp;4. Lalu isi file bebas kedalam branch itu. + push  
Anggota 2 (Renaldi):  
&nbsp;&nbsp;&nbsp;&nbsp;1. Clone repo  
&nbsp;&nbsp;&nbsp;&nbsp;2. Buat branch baru didalam repo dengan format penamaan (nama/nim)  
&nbsp;&nbsp;&nbsp;&nbsp;3. Lalu isi file bebas kedalam branch itu. + push  
Buat laporan Langkah Langkah yang dilakukan + dokumentasi (screenshot).
