TUGAS PRAKTIKUM SO SESI 1

Tugas dikerjakan secara berkelompok 2 anggota, boleh 3 anggota (maks) kalau satu orang tidak ada timnya. \n
Anggota 1 (Edo):
	1. Buat repo baru, nama kan sesuai nama kelompok
	2. Tambahkan kolaborator repo untuk teman tim
	3. Buat branch dengan format penamaan (nama/nim)
	4. Lalu isi file bebas kedalam branch itu. + push
Anggota 2 (Renaldi):
	1. Clone repo
	2. Buat branch baru didalam repo dengan format penamaan (nama/nim)
	3. Lalu isi file bebas kedalam branch itu. + push
Buat laporan Langkah Langkah yang dilakukan + dokumentasi (screenshot).
